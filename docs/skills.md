# 技能系统

NovaClaw 的技能系统允许您扩展 Agent 的能力，通过技能包实现自定义工作流。

## 技能包格式

技能包采用目录形式存放，每个技能对应一个目录：

```
skills/
└── skill-name/           # 技能目录，目录名为技能名称
    └── SKILL.md          # 技能定义文件（必需）
```

## SKILL.md 格式

`SKILL.md` 是技能的核心配置文件，采用 Markdown 格式：

```markdown
---
name: "技能名称"
description: "技能描述"
version: "1.0.0"
---

技能详细说明文档...
```

### 导入技能

NovaClaw 支持两种方式添加技能：

**方式一：通过前端界面导入**

1. 点击左侧菜单的 "技能" 图标
2. 点击 "添加技能" 按钮
3. 选择 ZIP 格式的技能包
4. 系统自动解压并导入技能

**方式二：手动复制到技能目录**

1. 将技能包 ZIP 文件解压
2. 将解压后的技能文件夹复制到技能存放目录
3. 系统会自动扫描并加载新技能

**技能存放路径**：
- **Windows**: `%USERPROFILE%\Documents\novaclaw\skills\`
- **macOS**: `~/Library/Application Support/novaclaw/skills/`
- **Linux**: `~/.local/share/novaclaw/skills/`

### 启用/禁用技能

1. 在技能列表中找到目标技能
2. 点击开关按钮切换状态
3. 禁用的技能不会被 Agent 调用

### 删除技能

1. 在技能列表中找到目标技能
2. 点击删除按钮
3. 确认删除操作

## 技能使用

Agent 会根据上下文自动选择合适的技能：

1. 用户提问时，Agent 分析需求
2. 匹配最合适的技能
3. 调用技能执行任务
4. 返回执行结果

## 创建自定义技能

### 步骤

1. 创建技能目录
2. 编写 `SKILL.md` 配置文件
3. 添加说明文档
4. 复制到技能存放目录或通过前端上传导入

### SKILL.md 示例

```markdown
---
name: "数据分析助手"
description: "帮助用户进行数据分析和可视化"
version: "1.0.0"
---

## 功能介绍

该技能可以帮助用户：
- 读取和解析 Excel 文件
- 进行数据清洗和转换
- 生成数据可视化图表
- 提供数据分析报告

## 使用方法

只需向 Agent 提出数据分析相关的请求即可。
```
