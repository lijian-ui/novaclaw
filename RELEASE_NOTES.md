### 概述

Jeeves 是一款基于 ReAct（推理-行动循环）架构的 AI Agent 桌面应用，采用 Rust 后端 + React 前端的全栈架构，通过 Tauri 2.0 打包为原生桌面应用。支持多模型提供商、MCP 协议扩展、技能系统、定时任务等企业级特性。

---

### 核心特性

- **🧠 ReAct Agent 引擎** — 思维链（CoT）推理过程可视化，工具调用过程透明可追溯
- **🔌 MCP 协议支持** — 连接任意 MCP 服务器，动态扩展 Agent 能力边界
- **🛠️ 20+ 内置工具** — 文件操作（读写/搜索/替换）、终端命令执行、网络搜索、代码搜索等
- **📦 技能系统** — ZIP 格式技能包，支持导入/导出/分享，可动态加载
- **⏰ 定时任务引擎** — 标准 Cron 表达式，支持后台自动执行与多平台投递
- **💬 多智能体协作** — Orchestrator + Worker 架构，支持子 Agent 管理与任务委派
- **🎨 现代化 UI** — 可拖拽分栏布局、Markdown/Mermaid 渲染、暗色主题、响应式设计
- **🔐 安全沙箱** — 工具执行权限策略、提示词注入防护
- **🌐 IM 网关集成** — 支持钉钉等 IM 平台的消息接入与自动回复

### 更新内容

#### Agent 引擎

- 实现标准 ReAct 推理-行动循环
- 支持多轮工具调用与中间推理
- 上下文窗口管理与 Token 预算控制
- 会话压缩（Compaction）机制，长对话自动摘要
- 流式输出支持，实时推送推理过程和工具调用

#### 多模型支持

- 兼容所有 OpenAI 格式 API（DeepSeek、智谱 GLM、LM Studio、Ollama 等）
- 模型上下文窗口大小自动获取与前端展示
- DeepSeek 缓存命中率统计与前端渲染

#### 工具系统

- 内置工具：文件读写、代码搜索（grep/glob）、终端命令、网络搜索
- 工具参数自动解析与路径归一化
- 工具执行结果实时流式推送
- MCP 工具动态注册与调用

#### 前端界面

- 聊天面板：Markdown 渲染、代码高亮、Mermaid 图表、思维链折叠展示
- 上下文用量环形进度条
- 缓存命中率展示（支持 DeepSeek 等模型）
- 会话管理：新建/切换/删除
- 模型选择与默认模型持久化
- 多语言支持（i18n）

#### IM 网关

- 钉钉平台接入（WebSocket 长连接）
- 微信个人号接入（WebSocket 长连接）
- IM 消息自动转 Agent 会话
- 群聊 @ 响应与私聊支持
- 跨平台会话持久化

#### 其他

- SOUL.md 智能体身份系统，可自定义 Agent 行为
- 定时任务引擎，支持 cron 表达式
- 技能系统，支持 ZIP 格式技能包
- 记忆系统，持久化用户偏好

### 安装说明

#### Windows

1. 下载最新版本的安装包（`.msi` 或 `.exe`）
2. 双击运行，按照安装向导完成安装
3. 启动 Jeeves，首次使用需配置 LLM API

#### macOS

1. 下载最新版本的 DMG 安装包
2. 打开 DMG 文件，将 Jeeves.app 拖入 Applications 文件夹
3. 首次启动时，若提示"无法验证开发者"，需前往 **系统设置 → 隐私与安全性**，点击"仍要打开"
4. 启动 Jeeves，首次使用需配置 LLM API

#### 配置文件

- 路径：`我的文档/Jeeves/config.json`（桌面版）
- 配置项：API Key、Base URL、模型选择、工具权限等
- 详细配置说明请参考 [配置文档](docs/configuration.md)

---

### 注意事项

1. **首次使用**：启动后需先配置 LLM 提供商（支持 DeepSeek、智谱 GLM、OpenAI 等）
2. **模型选择**：推荐使用支持 Prompt Caching 的模型以获得更好体验（如 DeepSeek V4）
3. **网络连接**：后端默认监听 `127.0.0.1:3000`，桌面版自动管理后端进程
4. **数据存储**：会话数据存储在 `我的文档/Jeeves/sessions/` 目录下，以 JSONL 格式保存

### 已知问题

- 前端缓存命中率在涉及工具调用或多轮推理的对话中偶有显示缺失（已修复，需后续版本验证）
- 部分 MCP 服务器存在兼容性问题，建议使用标准 MCP 协议实现
- 会话压缩后上下文缓存会完全失效（设计如此）
